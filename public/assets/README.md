# Assets

Déposez ici le vrai fichier logo :

```
proludik_h_rouge_blanc.png
```

(version horizontale rouge/blanc, texte blanc, conçue pour fond sombre).

Le header du dashboard (`public/index.html`) référence déjà ce chemin
exact. Tant que le fichier n'est pas présent, un logo de repli (texte
stylisé "PROLUDIK" en CSS/SVG) s'affiche automatiquement — dès que le
vrai fichier est commité à cet emplacement, il prend le relais sans
aucune modification de code.
